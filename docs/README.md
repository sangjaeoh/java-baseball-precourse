# 구현 기능 목록

- 규칙 클래스
  - [x] 3자리 수의 유효성 검사
  - [x] 3자리 수의 0, 같은 수 존재 여부 확인
  - [x] 컴퓨터, 사용자 3자리 수의 스트라이크 갯수 반환
  - [x] 컴퓨터, 사용자 3자리 수의 볼 갯수 반환
- 랜덤 숫자 생성 클래스
  - [x] 1부터 9까지 서로 다른 3자리의 수 생성
- 심판 클래스
  - [x] 스트라이크, 볼, 낫싱 여부 확인
  - [x] 스트라이크, 볼, 낫싱 프린트
  - [x] 컴퓨터, 사용자 3자리 수의 결과 심판 및 게임 종료 여부 확인
- 인풋 클래스
  - [x] 사용자에게 3자리 수 입력 받기

- [ ] 게임 종료 프린트
- [ ] 게임 새로 시작, 종료 입력 받기
